Temperature Converter

Це Python-додаток, який дозволяє конвертувати температуру між одиницями: Цельсій, Фаренгейт та Кельвін. Програма містить набір функцій та консольний інтерфейс.

Інструкції з встановлення

  1. Клонуйте репозиторій: 
    git clone https://github.com/Kreepier/temperature-converter.git
  2. Перейдіть у каталог: 
    cd temperature-converter
  3. Запустіть програму: 
    python converter.py

Приклади використання

from converter import celsius_to_fahrenheit, fahrenheit_to_kelvin

print(celsius_to_fahrenheit(25))   # 77°F
print(fahrenheit_to_kelvin(32))    # 273.15K

Опис функцій

Функція Опис Приклад
celsius_to_fahrenheit(c) Конвертує Цельсій у Фаренгейт celsius_to_fahrenheit(25)
fahrenheit_to_celsius(f) Конвертує Фаренгейт у Цельсій fahrenheit_to_celsius(32)
celsius_to_kelvin(c) Конвертує Цельсій у Кельвін celsius_to_kelvin(25)
kelvin_to_celsius(k) Конвертує Кельвін у Цельсій kelvin_to_celsius(300)
fahrenheit_to_kelvin(f) Конвертує Фаренгейт у Кельвін (через Цельсій) fahrenheit_to_kelvin(32)
