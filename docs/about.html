Проєкт Temperature Converter

Цей проєкт створений для конвертації температур між різними шкалами: Цельсій, Фаренгейт та Кельвін. Він містить набір функцій для математичних обчислень та консольний інтерфейс взаємодії з користувачем.

Мета проєкту

  • Навчитися працювати з Python-функціями
  • Закріпити навички роботи з Git та GitHub
  • Створити документацію у HTML-форматі
  • Організувати структуру файлів для майбутніх проєктів

Що всередині?

Проєкт складається з таких файлів:

  • converter.py – основний код програми
  • DOCUMENTATION.md – опис усіх функцій
  • README.md – опис проєкту та інструкції
  • index.html – документація у веб-форматі
